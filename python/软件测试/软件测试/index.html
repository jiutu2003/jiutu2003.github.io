1.项目用例(去哪了旅行网站订票系统)

用例ID 模块名 测试的功能 输入 测试步骤 预期结果 级别
qunaer_id_01 火车订票 预订火车票 1.打开火车票查询网站。2.输入正确的出发站和到达站。3.选择正确的车次。4.输入正确的乘客信息 1.正确打开网站。2.信息正确输入。3.车次选择正确。4.乘客信息正确输入 高
qunaer_id_02 火车订票 预订火车票 1.打开火车票查询网站。2.输入错误的出发站和到达站。3.选择正确的车次。4.输入正确的乘客信息 1.正确打开网站。2.信息无法输入，或输入不正确，无法进行下一步操作。 中
qunaer_id_03 火车订票 预订火车票 1.打开火车票查询网站。2.不输入出发站和到达站。3.选择正确的车次。4.输入正确的乘客信息 1.正确打开网站。2.信息没有输入，无法进行下一步操作。 低
